# Visão Geral

Este projeto tem como objetivo desenvolver um portfólio profissional para apresentar minha trajetória, habilidades técnicas e projetos desenvolvidos ao longo da minha formação.

Além da implementação da interface, este repositório demonstra um fluxo de trabalho semelhante ao utilizado em empresas de tecnologia, incluindo:

- Planejamento utilizando Scrum
- Gerenciamento de tarefas com Jira
- Versionamento com Git e GitHub
- Desenvolvimento incremental por Sprints
- Documentação da evolução do projeto

---

# Objetivos

- Desenvolver um portfólio moderno e responsivo
- Aplicar HTML5 semântico
- Utilizar CSS moderno para construção da interface
- Implementar funcionalidades com JavaScript
